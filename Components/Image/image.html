Responsive Images

Responsive Image Components are used to dsiplay Images on websites, which are screen responsive. They expand upto the width of the container they are in.

Square Image

Square Images are the square shaped images with rounded corners.

Round Image

Round Images are the circular Images

Full Width Images

Full Width Image is the one that expands to cover the container it is present in. Resize the browser window, to see the responsive property
